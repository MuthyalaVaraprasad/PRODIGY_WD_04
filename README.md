# ✨ Premium Visual Features

### 🌈 1. Dual-Theme Canvas Background Flow
- **Dark Mode (3D Cyber-Grid):** Renders a perspective holographic grid in neon violet and cyan that shifts based on mouse movement, accompanied by floating data packets cascading down.
- **Light Mode (Liquid Silk Waves):** Renders 3 layers of smooth pastel gradient wave ribbons that wave slowly and distort interactively based on cursor coordinates.

### 🎯 2. Custom Glowing Fluid Cursor
- Includes a desktop pointer setup carrying a solid center dot and a lag-following outer circle.
- Highlights mouse hovers with scale-expansion overlays and flashes a pink click-ripple upon presses.

### 🌀 3. 3D Perspective Scroll & Portal Transitions
- Homepage sections load with a spring scale-and-tilt perspective entry as you scroll.
- Clicking "Live Demo" on project cards triggers a full-screen zoom warp portal transition, sliding the viewport through spinning neon rings before redirecting to the target URL.

### 📊 4. Interactive Developer Dashboards
- **AI Career Readiness Auditor:** Pastes target job descriptions, computes compatibility indices (ATS %), and outlines learning roadmaps for skill gaps.
- **AI Interview Simulator:** Flips flashcards to reveal correct conceptual responses for React, SQL, and general coding questions.
- **DevOps Health Monitor:** Pings URL endpoints dynamically to display online status badges and latency speeds.
- **Retro Gamified Sandbox:** A custom Snake game themed around coding modules (eat React, JS, or Python to grow), carrying retro terminal outputs and mobile-friendly D-pads.

# 🤖 Nova AI Assistant Chatbot
- Floating chatbot panel preloaded with my complete resume knowledge.
- Supports query checks in **English, Telugu, and Hindi** (replies in the respective language).

# 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/MuthyalaVaraprasad/PRODIGY_WD_04.git
cd PRODIGY_WD_04
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm start
```
*App will launch on [http://localhost:3000](http://localhost:3000)*

### 4. Create production build
```bash
npm run build
```
