# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/MuthyalaVaraprasad/PRODIGY_WD_04.git
```

Navigate to the project

```bash
cd PRODIGY_WD_04
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
